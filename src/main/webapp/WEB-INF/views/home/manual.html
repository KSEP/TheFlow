User manual

  1. New company registration

    • If you already have your own company registered, or someone added you to his company, you can enter your cabinet by pressing button "Cabinet" on the start page and entering your username and password, which is the same for all companies you are related to. Press "Add company" button from your cabinet to add new company.
    • If you are a new user to the system, press "Registration button" on the start page. You will be redirected to the form, where you need to input your registration data. The new company with the new admin user will be created.

    During registering a new company, you have to input your company name and company alias – short one-word name with only letters and digits allowed.

  2. Signing in

    You have two ways to sign in:
    • Enter your own cabinet by pressing "Cabinet" button on the start page. You will see all companies you are related to, and you can simply login any of them, by pressing "Sign in" button at the end of the row with the company.
    • Enter "Login button" on the start page. Here you can login to a specific company. You need to input company alias here.

  3. Your cabinet

    You can access your own cabinet by pressing "Cabinet" button on the start page.

    In your cabinet you can see the list of companies you are registered in with your security role in every company. Also you can sign in in every company you see by pressing "Sign in" button

    Also you can manage companies which you created. To manage own companies press the button "Own companies" om the cabinet page.

  4. Starting to work

    After you have signed in you are on the home page. You can change your password on your profile page in the top right corner of the page.

    It has to be at least one project created to create new issue. Only admin users can create new projects. You can find your security role on your profile page.

  5. Creating a new project (is available only for admin users)

    You can create a new project from the home page, after you have signed in. Press "New project" button to create a new project.

    Also, you can create a new project from Admin tools → Manage projects →Add project

  6. Adding users to your company (is available only for admin users)

    You can add a new user to your company by following Admin tools → Manage users →Add user. You need to input new user's credentials. If the user with input email is already exists, system will alert you and you will be able to add existing user to your company with special form. User role will be assigned to added users in scope of your company.

  7. Managing users and projects (is available only for admin users)

    To add/remove/update users or projects go to Admin tools on the top of the home page.
