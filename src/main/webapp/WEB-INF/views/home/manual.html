User manual

New company registration

If you have been registered in the application already, you can enter your cabinet by pressing button "Cabinet" on the start page with entering your username and password, which is the same for all companies you are related to. Press the "Add company" button on your cabinet page to add a new company.

If you are a new user to the system, press the "Registration" button on the start page. You will be redirected to the form, where you need to input your registration data. The new company with the new admin user will be created.

During registering a new company, you have to input your company name and company alias – short one-word name with only letters and digits allowed.

Signing in

Enter "Login button" on the start page. Here you can login to a specific company. You need to input company alias.

When you are logged in you can go to the "Companies" page by pressing the "Companies" button on your profile page. Here you can see all companies you are registered in. It is possible to perform one-click login to any of this companies by pressing the "Sign in" button;

Managing companies

You can access your "Companies" page by pressing the "Companies" button on your profile page.

On your "Companies" page you can see the list of companies you are registered in with your security role in every company. Also you can sign in in every company you see by pressing the "Sign in" button

Also you can manage(add/edit/remove) companies which you have created. To manage own companies press the button "Own companies" on the "Companies" page.

Starting to work

After you have signed in you are on the main page. You can change your password on your profile page in the top right corner of the page header.

It has to be at least one project created to create a new issue. Only admin users can create new projects. You can find your security role on your profile page.

Creating a new project (is available only for admin users)

You can create a new project from the main page, after you have signed in. Press the "New project" button to create a new project.

Also, you can create a new project from Admin tools → Manage projects →Add project

Adding users to your company (is available only for admin users)

You can add a new user to your company by following "Admin tools" → Manage users →Add user. You need to input new user's credentials. If the user with input email is already exists, system will alert you and you will be able to add existing user to your company with special form. "User" role will be assigned to added users in scope of your company.

Managing users and projects (is available only for admin users)

To add/remove/update users or projects go to the "Admin tools" on the application header.
